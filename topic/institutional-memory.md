---
title: Institutional Memory
---
The post emphasizes that institutions need durable ways to store and transmit memory, rather than relying on individuals to remember critical exceptions, processes, and context. When memory moves from archive to veteran, the organization becomes fragile and person-dependent. This kind of implicit memory may work temporarily, but it cannot scale or survive turnover. The text connects this to coordination and legitimacy, showing that memory is part of the deeper layer that makes institutions livable. A healthy institution distributes memory across structures instead of concentrating it in a few overburdened people.

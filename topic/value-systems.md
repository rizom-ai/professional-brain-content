---
title: Value Systems
---
Value systems are a central concern in the work on Public Badges, trust, governance, and collaborative tooling. The content argues that software and organizations are never value-neutral, even when those values are implicit. Several pieces propose making values explicit, measurable, and communicable so that users and participants can align around them. Public Badges is a key example, turning ethical commitments into visible evidence rather than vague claims. The broader argument is that trust grows when values are articulated and negotiated, not when security alone is imposed. This topic connects ethics, certification, and organizational design into a single practical problem of alignment.

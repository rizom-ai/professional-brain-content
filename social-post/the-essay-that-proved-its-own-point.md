Today, an essay I wrote about institutional memory disappeared from my website.

The essay was called *Hiding in Plain Sight*.

The irony is almost too perfect.

The essay had been published, indexed, and shared. The content still existed. The Git history still existed. The images, links, and generated pages still existed.

But one durable record still called it a draft.

After a deployment rollback and a full synchronization, the system trusted that stale record. The published essay quietly disappeared.

Nothing was lost. Everything was stored. But the traces no longer added up to memory.

That is exactly the argument of the essay: institutions confuse storage with memory.

Storage preserves evidence. Memory preserves orientation—the context needed to understand what happened, what still matters, and what should happen next.

We like to call something a “source of truth,” as if authority guarantees accuracy. But a source of truth can still be wrong. It merely decides which version of reality wins when the system has to reconcile itself.

Sometimes an institution doesn’t forget because its records are missing.

Sometimes it remembers the wrong thing with perfect confidence.
